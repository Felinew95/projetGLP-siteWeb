# Projet Génie Logiciel Projet - Site Web

*Un site dédié au projet Génie Logiciel Projet de l'Équipe B-Histoire*

Ce site présente une **simulation interactive de l’histoire de la civilisation viking**. Les Vikings ont vécu en Scandinavie (Norvège, Suède) entre 793 et 1066, connus pour leurs **explorations, commerces et raids** à travers l’Europe. La simulation permet de découvrir **l’évolution politique, économique, militaire et culturelle** de cette civilisation sur plusieurs siècles.
